# 법무 올인원 대시보드 (Law Firm All-in-One Dashboard)

업무 활용도를 높이기 위한 단일 HTML 대시보드입니다. 서버 없이 `index.html`을 브라우저로 열면 바로 사용할 수 있습니다.

## 주요 기능
- **하늘색 테마** 대시보드 + 실시간 시계/날짜/인사말
- **GitHub 올인원** 바로가기 (저장소·이슈·PR·커밋·알림 등)
- **법률 업무 도구** 링크 (나의 사건검색·전자소송·종합법률정보·국가법령정보·케이스노트·인터넷등기소 등)
- **업무 메모(간략메모) → 오늘 할일 자동 보고**
  - 순번 아이콘 ①②③ 정렬
  - 5글자 상태 보고: `처리대기중 → 처리진행중 → 처리완료됨`
  - 진행도 표시(아침 체크)
  - Google 캘린더 추가 / `.ics` 일괄 저장
- 메모·할일 데이터는 브라우저(localStorage)에만 저장

## 파일
| 파일 | 설명 |
|------|------|
| `index.html` | 대시보드 본체 |
| `gavel.ico` / `gavel-icon.svg` / `gavel-icon.png` | 판사봉 아이콘 |
| `wallpaper-skyblue.png` | 하늘색 바탕화면 |
| `setup-desktop.ps1` | 바탕화면 적용 + 바로가기 생성 |
| `make-assets.ps1` | 아이콘/배경 재생성 스크립트 |

## 사용법
1. `index.html` 더블클릭 → 브라우저로 열기
2. (선택) `setup-desktop.ps1` 우클릭 → PowerShell로 실행 → 바탕화면 적용 + 바로가기 생성

> 참고: 개인 PC에서 사용하는 일부 아이콘/배경 이미지는 제3자 저작권 보호를 위해 저장소에서 제외되어 있습니다(`.gitignore`). 저장소에서는 판사봉 아이콘으로 표시됩니다.
